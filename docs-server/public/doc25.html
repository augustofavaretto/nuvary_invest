🖨️ PDF Fiscal na Aba Imposto de Renda

Dois documentos gerados via window.print() — sem dependências externas

A aba Imposto de Renda da página /relatorios ganhou dois botões de download de documentos fiscais, exatamente como os bancos geram para declaração do IR.

A geração de PDF usa window.open() + template HTML inline + window.print() — zero dependências adicionais (sem jsPDF, sem html2canvas). O usuário recebe um diálogo "Imprimir" nativo do browser e pode salvar como PDF.
🔴 DARF — Código 6015
  • •Cabeçalho oficial: "DOCUMENTO DE ARRECADAÇÃO DE RECEITAS FEDERAIS"
  • •Código de receita: 6015 — Ganhos Líquidos em Operações em Bolsa
  • •Período de apuração, vencimento e valor calculado (lucro × 15%)
  • •Número do CPF/CNPJ, nome e competência
  • •Aviso: "Não é documento oficial — para fins de planejamento"
🔵 Informe de Rendimentos
  • •Cabeçalho: "INFORME DE RENDIMENTOS — Ano-Calendário"
  • •4 seções: Renda Fixa, Renda Variável, Dividendos e FIIs, Resumo IR
  • •Valores extraídos da carteira real do Supabase (portfolioService)
  • •Lucro tributável, alíquota 15%, IR devido e isenções PF (≤ R$20.000)
  • •Rodapé com instruções de declaração no IRPF

🚪 Detalhes do Fix de Logout

Antes × Depois — DashboardLayout.tsx

❌ Antes

const handleLogout = async () => { await logout(); router.push('/login'); // não funcionava };

O router.push do Next.js não forçava recarga completa — o estado de auth persistia em memória.

✅ Depois

const handleLogout = async () => { await logout(); window.location.href = '/'; // hard reload };

window.location.href força recarga completa da página — auth state é limpo, usuário vai para a landing page.

🎨 Contraste do Questionário — Dark Mode

3 componentes atualizados — QuestionCard, Questionnaire, ResultCard

O questionário de perfil de risco usava cores hardcoded compatíveis apenas com tema claro. Com o dark mode ativo, todo o texto ficava invisível.

Antes (hardcoded, light only) Depois (CSS var, responsivo) Uso
text-[#0B1F33] text-foreground Títulos e textos principais
text-[#6B7280] text-muted-foreground Textos secundários e labels
border-[#E5E7EB] border-border Bordas de cards e opções
bg-[#E5E7EB] / bg-[#F3F4F6] bg-muted Fundo progress bar, letras inativas
bg-white (implícito) bg-card Fundo dos cards de pergunta
As cores de marca (#00B8D9, #007EA7, gradientes de perfil) foram mantidas — apenas as cores utilitárias que quebravam no dark mode foram substituídas.
