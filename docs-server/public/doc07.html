Visao Geral

O Chatbot de Investimentos da Nuvary Invest e um assistente virtual alimentado por OpenAI que oferece suporte personalizado aos usuarios. Integrado ao perfil do investidor, o chatbot fornece respostas contextualizadas sobre investimentos, analises de mercado e educacao financeira.

Principais Caracteristicas

  • Integracao com OpenAI - Respostas inteligentes usando GPT
  • Contexto do Perfil - Personalizacao baseada no perfil do investidor
  • Historico de Conversa - Mant√©m contexto das ultimas mensagens
  • Acoes Rapidas - Botoes pre-definidos para perguntas comuns
  • Interface Moderna - Design conversacional com animacoes
  • Responsivo - Funciona em desktop e mobile

Contexto Inteligente

Perfil do Investidor

O chatbot recebe automaticamente o perfil do usuario (tipo, pontuacao, alocacao recomendada) e personaliza todas as respostas com base nessas informacoes.

Historico de Conversa

As ultimas 10 mensagens sao enviadas como contexto, permitindo conversas continuas e referencias a perguntas anteriores.

Chatbot.tsx

Componente principal que gerencia todo o fluxo do chat:

  • Estado das mensagens e loading
  • Carregamento do perfil do localStorage
  • Comunicacao com API do backend
  • Scroll automatico para novas mensagens
  • Mensagem de boas-vindas personalizada

ChatMessage.tsx

Renderiza cada mensagem individual:

  • Avatar diferenciado (usuario vs assistente)
  • Bolha com estilo apropriado
  • Indicador de digitacao animado
  • Timestamp da mensagem
  • Animacoes com Framer Motion

QuickActions.tsx

Grid de botoes de acao rapida:

  • 8 acoes pre-definidas
  • Icones do Lucide React
  • Animacao de entrada escalonada
  • Dispara prompt ao clicar

API Backend

Endpoint Principal

POST /api/ai/chat

Envia mensagem e recebe resposta do assistente

Request Body

{ "message": "Qual a alocacao ideal para meu perfil?", "conversationHistory": [ { "role": "user", "content": "Ola" }, { "role": "assistant", "content": "Ola! Como posso ajudar?" } ] }

Response

{ "success": true, "content": "Para o seu perfil Moderado, a alocacao recomendada e...", "usage": { "prompt_tokens": 150, "completion_tokens": 200, "total_tokens": 350 } }

Outros Endpoints de IA

POST /api/ai/ask

Pergunta simples sem historico

GET /api/ai/explain/:term

Explica termo financeiro

POST /api/ai/suggestion

Sugestao de investimento baseada no perfil

POST /api/ai/analyze/stock/:symbol

Analisa uma acao especifica

POST /api/ai/market-summary

Resumo geral do mercado

Integracao com Perfil

Fluxo de Dados

1. Usuario completa questionario ‚Üì 2. Perfil salvo no localStorage ('investorProfile') ‚Üì 3. Chatbot carrega perfil no useEffect ‚Üì 4. Contexto adicionado a cada mensagem enviada ‚Üì 5. Backend recebe perfil + mensagem ‚Üì 6. OpenAI gera resposta personalizada

Se o usuario nao tiver perfil cadastrado, o chatbot exibe um card incentivando a realizacao do questionario para obter respostas personalizadas.
