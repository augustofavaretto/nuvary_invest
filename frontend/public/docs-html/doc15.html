Pagina Minha Carteira

A pagina "Minha Carteira" apresenta uma visao completa do portfolio de investimentos, inspirada no portal do investidor B3. Inclui resumo patrimonial, graficos de distribuicao e detalhamento por tipo de ativo.

Saldo no Dashboard Header

O patrimonio total tambem e exibido no header do dashboard, ao lado do nome do usuario, com botao para ocultar/mostrar o valor. A preferencia e salva no localStorage.

Resumo Patrimonial

Card com gradiente escuro mostrando patrimonio total, valor investido, lucro/prejuizo e rentabilidade.

Distribuicao por Classe

Grafico donut com distribuicao entre Renda Fixa, Renda Variavel, FIIs e Internacional.

Distribuicao por Produto

Tabs com lista de ativos por categoria, incluindo ticker, quantidade, preco medio e variacao.

Distribuicao por Corretora

Grafico donut mostrando quanto esta alocado em cada corretora (XP, Clear, Rico, etc).

Graficos Donut (Recharts)

Os graficos de pizza/donut sao renderizados com a biblioteca Recharts, oferecendo animacoes suaves, tooltips interativos e design responsivo.

Interface PortfolioData

export interface PortfolioSummary {
  totalValue: number;      // Patrimonio total
  totalInvested: number;   // Valor investido
  totalProfit: number;     // Lucro/Prejuizo
  profitPercentage: number; // Rentabilidade %
  lastUpdate: string;      // Data atualizacao
}

export interface Asset {
  ticker: string;
  name: string;
  quantity: number;
  averagePrice: number;
  currentPrice: number;
  totalValue: number;
  variation: number;
  class: AssetClass;
}

export interface PortfolioData {
  summary: PortfolioSummary;
  byClass: { name: string; value: number; color: string }[];
  byProduct: {
    rendaFixa: Asset[];
    rendaVariavel: Asset[];
    fiis: Asset[];
    internacional: Asset[];
  };
  byBroker: { name: string; value: number; color: string }[];
}

Toggle de Visibilidade do Saldo

// DashboardHeader.tsx
const [showBalance, setShowBalance] = useState(true);

useEffect(() => {
  const savedPreference = localStorage.getItem('nuvary_show_balance');
  if (savedPreference !== null) {
    setShowBalance(savedPreference === 'true');
  }
}, []);

const toggleBalance = () => {
  const newValue = !showBalance;
  setShowBalance(newValue);
  localStorage.setItem('nuvary_show_balance', String(newValue));
};
