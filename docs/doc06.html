Visao Geral

O frontend da Nuvary Invest foi desenvolvido utilizando as tecnologias mais modernas do ecossistema React, seguindo as diretrizes do manual de marca e priorizando performance, acessibilidade e experiencia do usuario.

Funcionalidades Implementadas

  • Landing Page - Pagina inicial com apresentacao da plataforma
  • Questionario de Perfil - Interface interativa para o questionario de 10 perguntas
  • Tela de Resultado - Exibicao detalhada do perfil de investidor
  • Animacoes Fluidas - Transicoes suaves com Framer Motion
  • Design Responsivo - Layout adaptavel para mobile e desktop
  • Identidade Visual - Cores, tipografia e logos conforme manual da marca

Stack Tecnologico

O projeto utiliza as versoes mais recentes das bibliotecas, garantindo performance e compatibilidade.

Next.js 16 + React 19

Framework React com App Router, Server Components e otimizacoes automaticas

TypeScript

Tipagem estatica para maior seguranca e produtividade no desenvolvimento

Tailwind CSS v4

Framework CSS utility-first com variaveis CSS nativas e tema customizado

shadcn/ui + Radix UI

Componentes acessiveis e customizaveis: Button, Card, Badge, RadioGroup, Label, Progress

Framer Motion

Animacoes declarativas para transicoes de pagina, opcoes e resultados

Lucide React

Biblioteca de icones SVG consistentes e leves

Componentes do Questionario

Questionnaire

src/components/questionnaire/Questionnaire.tsx

Componente principal que gerencia o fluxo do questionario: introducao, perguntas, loading e resultado.

  • Gerencia estado das telas (intro, questions, loading, result)
  • Busca perguntas da API no mount
  • Envia respostas e recebe perfil classificado
  • Barra de progresso animada

QuestionCard

src/components/questionnaire/QuestionCard.tsx

Exibe uma pergunta individual com suas opcoes de resposta.

  • Badge de categoria com icone (Objetivos, Horizonte, Tolerancia)
  • RadioGroup acessivel para selecao
  • Animacoes de entrada/saida com Framer Motion
  • Feedback visual na opcao selecionada

ResultCard

src/components/questionnaire/ResultCard.tsx

Exibe o resultado completo do questionario com perfil e recomendacoes.

  • Header com gradiente do perfil e icone
  • Analise por categoria com barras animadas
  • Grafico de alocacao recomendada
  • Lista de investimentos sugeridos
  • Caracteristicas do perfil
  • Botao para refazer questionario

Executando o Projeto

Pre-requisitos

  • Node.js 18+ instalado
  • Backend rodando na porta 3001

Instalacao

# Navegar para o diretorio cd frontend-next # Instalar dependencias npm install # Iniciar em desenvolvimento npm run dev

URLs

Servico URL
Frontend http://localhost:3000
Questionario http://localhost:3000/questionario
Backend API http://localhost:3001/api

Variavel de Ambiente
