Codigo - authService.ts

export async function loginWithGoogle() {
  const { data, error } = await supabase.auth.signInWithOAuth({
    provider: 'google',
    options: {
      redirectTo: `${window.location.origin}/questionario`,
      queryParams: {
        access_type: 'offline',
        prompt: 'consent',
      }
    }
  });

  if (error) throw error;
  return data;
}

Novos Campos de Cadastro

1. CPF

CPF *
123.456.789-09
  • Mascara: 000.000.000-00
  • Validacao: Algoritmo modulo 11 oficial
  • Armazenamento: Somente numeros no banco

2. Data de Nascimento

Data de Nascimento *
15/03/1990
  • Mascara: DD/MM/AAAA
  • Validacao: Data valida + idade minima 18 anos
  • Formato no banco: YYYY-MM-DD (ISO)

3. Telefone

Telefone *
(11) 99999-9999
  • Mascara celular: (00) 00000-0000
  • Mascara fixo: (00) 0000-0000
  • Validacao: 10 ou 11 digitos

Funcao de Validacao de CPF

function validarCPF(cpf: string): boolean {
  const numeros = cpf.replace(/\D/g, '');

  if (numeros.length !== 11) return false;
  if (/^(\d)\1+$/.test(numeros)) return false;

  // Calculo do primeiro digito verificador
  let soma = 0;
  for (let i = 0; i < 9; i++) {
    soma += parseInt(numeros[i]) * (10 - i);
  }
  let resto = (soma * 10) % 11;
  if (resto === 10 || resto === 11) resto = 0;
  if (resto !== parseInt(numeros[9])) return false;

  // Calculo do segundo digito verificador
  soma = 0;
  for (let i = 0; i < 10; i++) {
    soma += parseInt(numeros[i]) * (11 - i);
  }
  resto = (soma * 10) % 11;
  if (resto === 10 || resto === 11) resto = 0;
  if (resto !== parseInt(numeros[10])) return false;

  return true;
}
Algoritmo Modulo 11: O CPF brasileiro usa dois digitos verificadores calculados atraves do algoritmo modulo 11. Cada digito e calculado multiplicando os digitos anteriores por pesos decrescentes e aplicando o resto da divisao por 11.

Configuracao do Google OAuth

1. Google Cloud Console

  1. Acessar console.cloud.google.com
  2. Criar novo projeto ou selecionar existente
  3. APIs e Servicos > Credenciais
  4. Criar credenciais > ID do cliente OAuth
  5. Tipo: Aplicativo da Web
  6. Adicionar URIs de redirecionamento autorizados: 
    https://[PROJECT_ID].supabase.co/auth/v1/callback

2. Supabase Dashboard

  1. Authentication > Providers
  2. Encontrar Google na lista
  3. Habilitar o toggle
  4. Colar Client ID e Client Secret
  5. Salvar alteracoes
Importante: O erro "Unsupported provider: provider is not enabled" indica que o toggle do Google nao foi habilitado no Supabase Dashboard.
